# Class2Calendar Pricing Strategy

> **Note**: This document outlines the pricing structure and feature differentiation between Free and Pro tiers.

## Pricing Tiers

| Feature | Free | Pro |
| :--- | :--- | :--- |
| **Price** | **$0** | **$20** |
| **Sync Frequency** | 1 sync per 24 hours | 1 sync per 15 minutes |
| **Manual Sync** | No | Yes (Ability to sync manually) |
| **Customization** | Limited / No customization | Full customization |
| **Alerts** | No alerts/warnings | Alerts for high workload |
| **Platform** | Local only extension | Cross-device cloud support |

---

## Feature Details

### "Full Customization" Entails:
- **Smart Delegation**: Delegating events vs assignments into Calendar vs Tasks
- **Visuals**: Custom colors for different courses/types
- **Date Range**: Custom date syncing range (past/future)
- **Selection**: Selecting specific custom courses to sync
- **Naming**: Custom naming rules for courses/assignments

### Upsell Strategy
- Include a **"Last synced [HMS] hours ago"** indicator in the UI to entice free users to upgrade for real-time updates.

---

## Future Ideas for "Max" Tier

These features are planned for a potential higher tier or future update:

1.  **Analytics + Stats**: Usage tracking, completion history.
2.  **Task Prioritization**: Smart sorting of tasks.
3.  **Automatic Study/Work Block Insertion**: Blocking time in calendar before deadlines.
4.  **Syllabus Parsing**:
    - Helps task prioritization
    - Automatically extracts exam dates
5.  **AI Integration**: Summaries and assistants.
6.  **Better Sync**: Trigger sync on event change (if technically feasible).
